# Подробные планы лекций по дисциплине "Базы данных"

Этот каталог дополняет файл `DB/updated_database_course_program.md` и раскрывает содержание каждой лекции двухсеместрового курса.

Общая сетка:

- 5-й семестр: 16 лекций, фокус на фундаменте реляционной модели, SQL и прикладном использовании СУБД.
- 6-й семестр: 16 лекций, фокус на внутреннем устройстве, эксплуатации, современных архитектурах данных и инженерных кейсах сопровождения БД.

Каждый файл содержит:

- место лекции в курсе;
- цели занятия;
- подробный план пары;
- ключевые понятия;
- связь с практическим занятием;
- контрольные вопросы в формате "что вы должны были узнать за эту лекцию".
