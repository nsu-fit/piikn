# Ожидаемые свойства решения

В корректном решении:

- задачи 1-3 используют selection, projection и join без случайных декартовых произведений;
- задача 4 выражает отсутствие сегментов через разность или антисоединение;
- задача 5 выражает "для всех" через division или эквивалентную комбинацию операций;
- ручные результирующие таблицы совпадают с малыми отношениями из `provided/small_relations.md`;
- эквивалентность в задаче 6 объяснена через одинаковый набор строк результата.

SQL-запросы не требуются и не повышают оценку.
