# Контракт проверки

Автоматически можно проверить:

- наличие `README.md`, `answers.md`, `result_tables.md`;
- отсутствие обязательного `*.sql`;
- наличие ответов на задания 1-6;
- наличие результирующих таблиц для заданий 1, 2, 3 и 5.

Вручную проверяется:

- корректность выражений реляционной алгебры;
- корректность ручного вычисления;
- отсутствие случайного декартова произведения;
- понимание разности, антисоединения и деления.
