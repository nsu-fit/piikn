# Практика 5.5. Реляционная алгебра без самостоятельного SQL

На этой неделе вы работаете с выражениями реляционной алгебры и малыми табличными фрагментами. SQL писать не нужно.

## Исходные отношения

Используйте таблицы из `provided/small_relations.md`.

Дополнительные варианты для тренировки и индивидуализации заданий находятся в `provided/raw_derived_relational_algebra_variants.md`. Основное задание этой практики опирается на `provided/small_relations.md`; дополнительные варианты выдаются преподавателем отдельно.

Обозначения:

- selection: `sigma_{condition}(R)`;
- projection: `pi_{attributes}(R)`;
- join: `R join_{condition} S`;
- difference: `R - S`;
- division: `R ÷ S`.

Можно использовать Unicode-обозначения `σ`, `π`, `⋈`, `÷`, если вам так удобнее.

## Задания

### 1. Selection и projection

Запишите выражение, которое возвращает коды и города аэропортов из России.

Затем вручную вычислите результат на таблице `Airports`.

### 2. Join

Запишите выражение, которое возвращает `route_no`, город отправления и город прибытия для всех маршрутов.

Вручную вычислите результат для маршрутов `R1`, `R2`, `R3`.

### 3. Join + selection

Запишите выражение, которое возвращает пассажиров, летящих рейсом `F102`.

Результат должен содержать `ticket_no`, `passenger_name`, `flight_id`.

### 4. Отсутствие связанной строки

Запишите выражение, которое возвращает билеты, у которых нет ни одного сегмента.

В ответе поясните, почему это антисоединение или разность.

### 5. "Для всех"

Отношение `RequiredFlights` содержит рейсы, которые входят в обязательный маршрут проверки.

Запишите выражение, которое возвращает билеты, имеющие сегменты для всех рейсов из `RequiredFlights`.

### 6. Эквивалентность

Для задачи 3 предложите вторую эквивалентную форму выражения:

- сначала отфильтровать `Segments` по `flight_id = F102`, затем соединять;
- или сначала соединить `Tickets` и `Segments`, затем фильтровать.

Поясните, почему на данном наборе данных результаты совпадают.

## `answers.md`

Для каждого задания укажите:

- выражение реляционной алгебры;
- краткое пояснение смысла;
- если есть альтернативная форма, почему она эквивалентна.

## `result_tables.md`

Для заданий 1, 2, 3 и 5 вручную приведите результирующие таблицы.
