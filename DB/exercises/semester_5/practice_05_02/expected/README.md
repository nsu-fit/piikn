# Ожидаемые свойства решения

## schema_profile.sql

- Совпадает с выданным `provided/schema_profile.sql` или отличается только комментариями студента.
- Выполняется без ошибок.

## data_dictionary.md

- Описаны все таблицы варианта.
- Для каждой таблицы указан предметный смысл, ключи и важные ограничения.
- Есть раздел "Неявные или спорные правила".

## query_observations.md

- Есть разделы `columns`, `constraints`, `cardinality`, `quality checks`.
- Наблюдения относятся к таблицам варианта.
- Студент объясняет, что проверяет каждый выбранный диагностический запрос.

## answers.md

- Ответы различают ограничение схемы, наблюдение на данных и гипотезу предметной области.
