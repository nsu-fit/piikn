- Проверить наличие всех обязательных файлов.
- Выполнить `q01_columns.sql`, `q02_constraints.sql`, `q03_cardinality.sql`, `q04_quality_checks.sql`.
- Проверить, что `q01_columns.sql` и `q02_constraints.sql` фильтруют таблицы согласно варианту.
