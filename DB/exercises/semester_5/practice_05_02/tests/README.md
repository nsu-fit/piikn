# Контракт проверки

## Автоматические проверки

- Проверить наличие всех обязательных файлов.
- Выполнить `schema_profile.sql`.
- Проверить, что сценарий содержит блоки `q01_columns`, `q02_constraints`, `q03`, `q04`.
- Проверить, что `query_observations.md` содержит обязательные разделы.

## Hidden checks

- На этой неделе hidden checks не используются для оценки SQL-навыка, потому что студенты не пишут SQL самостоятельно.
- Возможна проверка, что `schema_profile.sql` выполняется на базе `6 months`.

## Ручная проверка

- `data_dictionary.md`, `query_observations.md` и `answers.md` проверяются вручную.
- Особое внимание: студент не должен путать правило, доказанное ограничением, и гипотезу, замеченную по текущим данным.
