# Контракт автопроверки

Автопроверка запускается на видимой базе `3 months`; при необходимости те же SQL-файлы могут быть запущены на скрытой базе большего размера.

## Проверки файлов

- существует `README.md`;
- существуют `q01_environment.sql`, `q02_schema_objects.sql`, `q03_table_counts.sql`, `q04_smoke_queries.sql`, `answers.md`;
- SQL-файлы не пустые.

## SQL checks

- каждый SQL-файл выполняется без ошибок;
- `q01_environment.sql` возвращает одну строку;
- `q02_schema_objects.sql` содержит ожидаемые объекты схемы;
- `q03_table_counts.sql` возвращает обязательные таблицы и не содержит жестко заданных `row_count`;
- `q04_smoke_queries.sql` возвращает три result set или три явно разделенных запроса, которые можно выполнить по отдельности.

## Manual review

- `answers.md` проверяется вручную;
- студент должен уметь объяснить, почему запросы переносятся на базу другого размера.
