# Практика 5.1. Вход в Taidon/sqlrs и воспроизводимый SQL-сценарий

## Контекст

Вы работаете с демонстрационной базой Postgres Pro "Авиаперевозки", версия `2025-09-01`, схема `bookings`. В обычной практике используется вариант базы `3 months`.

Цель работы - не решить сложную предметную задачу, а проверить, что вы умеете:

- открыть учебную SQL-среду;
- выполнить SQL-файл;
- сохранить воспроизводимые запросы;
- зафиксировать результат в Markdown;
- не подменять SQL-артефакты скриншотами.

## Что нужно сдать

## Задание

### 1. Проверка окружения

В файле `q01_environment.sql` напишите запрос, который возвращает ровно одну строку со столбцами:

- `demo_version` - результат `bookings.version()`;
- `snapshot_time` - результат `bookings.now()`;
- `search_path` - значение `current_setting('search_path')`;
- `db_user` - текущий пользователь БД.

### 2. Объекты схемы

В файле `q02_schema_objects.sql` напишите запрос к системным каталогам PostgreSQL, который выводит объекты схемы `bookings`.

Результат должен иметь столбцы:

- `object_name`;
- `object_type`.

Нужно вывести таблицы, представления и последовательности. Результат отсортируйте по `object_type`, затем по `object_name`.

Подсказка: можно использовать `pg_catalog.pg_class` и `pg_catalog.pg_namespace`.

### 3. Размер основных таблиц

В файле `q03_table_counts.sql` напишите запрос, который возвращает количество строк в основных таблицах:

- `bookings`;
- `tickets`;
- `segments`;
- `flights`;
- `routes`;
- `boarding_passes`;
- `seats`;
- `airports_data`;
- `airplanes_data`.

Результат должен иметь столбцы:

- `table_name`;
- `row_count`.

Порядок строк: по убыванию `row_count`, затем по `table_name`.

Важно: не вписывайте ожидаемые количества вручную. Запрос должен считать строки в текущей базе.

### 4. Smoke queries

В файле `q04_smoke_queries.sql` напишите три запроса.

`q04a`: количество мест по модели самолета и классу обслуживания.

Столбцы:

- `airplane_code`;
- `fare_conditions`;
- `seat_count`.

`q04b`: 10 ближайших по расписанию рейсов относительно `bookings.now()`.

Столбцы:

- `flight_id`;
- `route_no`;
- `departure_airport`;
- `arrival_airport`;
- `scheduled_departure`;
- `status`.

Используйте `timetable`, сортировку по `scheduled_departure`, затем `flight_id`.

`q04c`: 10 стран с наибольшим числом аэропортов.

Столбцы:

- `country`;
- `airport_count`.

Используйте представление `airports`, а не `airports_data`, чтобы получить человекочитаемые названия.

- Не используйте hard-coded значения количества строк.
