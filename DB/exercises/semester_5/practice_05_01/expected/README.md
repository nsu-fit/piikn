# Ожидаемые свойства решения

## q01_environment.sql

- Возвращает одну строку.
- Обязательные столбцы: `demo_version`, `snapshot_time`, `search_path`, `db_user`.
- `demo_version` содержит строку версии демобазы `PostgresPro 2025-09-01`.

## q02_schema_objects.sql

- Возвращает объекты схемы `bookings`.
- Обязательные столбцы: `object_name`, `object_type`.
- В результате должны присутствовать как минимум:
  - `bookings`;
  - `tickets`;
  - `segments`;
  - `flights`;
  - `routes`;
  - `airports`;
  - `airplanes`;
  - `timetable`.

## q03_table_counts.sql

- Возвращает строку для каждой обязательной таблицы.
- `row_count` должен вычисляться запросом, а не задаваться литералом.
- На больших вариантах базы значения могут отличаться, но набор таблиц и порядок сортировки должны сохраняться.

## q04_smoke_queries.sql

- Содержит три запроса: `q04a`, `q04b`, `q04c`.
- Каждый запрос должен иметь указанные в задании столбцы.
- Для запросов с `LIMIT 10` обязателен детерминированный `ORDER BY`.
