# Ожидаемые свойства решения

## intro_queries.sql

- Совпадает с выданным `provided/intro_queries.sql` или отличается только комментариями студента.
- Выполняется без ошибок на базе `3 months`.
- Не содержит добавленных hard-coded результатов.

## run_log.md

- Указана версия демобазы.
- Отмечено, какие блоки сценария выполнены.
- Есть краткое описание результата, а не только скриншот.

## answers.md

- Ответы связаны с результатом выполнения готового SQL-сценария.
- Студент различает таблицы и представления.
