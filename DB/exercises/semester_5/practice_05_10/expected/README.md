# Ожидаемые свойства решения

В корректной работе:

- зависимости отделены от случайных совпадений в текущих строках;
- кандидатный ключ обоснован через смысл сегмента билета и замыкание атрибутов;
- SQL-проверки ищут противоречивые повторы, а не просто считают строки;
- аномалии описаны как сценарии вставки, обновления и удаления;
- черновик декомпозиции пригоден как вход в практику 5.11.

Не требуется создавать новую схему или переносить данные.
