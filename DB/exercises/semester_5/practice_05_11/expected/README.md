# Ожидаемые свойства решения

В корректном решении:

- нормализованные таблицы отражают зависимости из практики 5.10;
- `norm_segments` связывает билет и рейс составным ключом или эквивалентным уникальным ограничением;
- внешние ключи фиксируют связи между таблицами;
- перенос данных не создает дубликаты справочников;
- проверки `EXCEPT` показывают отсутствие потерянных и лишних строк.

Допускаются разные декомпозиции, если они явно объяснены и проходят проверку обратного соединения.
