# Ожидаемые свойства решения

В корректной работе:

- зависимости выводятся из текста предметной области;
- декомпозиция не теряет строки при обратном соединении;
- транзакционный timeline разобран по шагам;
- небезопасный SQL исправлен через параметры или whitelist;
- ответы достаточно краткие для контрольной, но содержат проверяемые рассуждения.

Не ожидается полноценный промышленный migration plan.
