# Ожидаемые свойства решения

В корректном решении:

- `schema.sql` создает схему `course_lab` и три требуемые таблицы;
- первичные и внешние ключи выражены явно;
- `code`, `status`, `priority` и `closed_at` защищены ограничениями;
- `external_key` уникален;
- `provided/check_constraints.sql` выполняется и показывает ожидаемые `NOTICE`;
- `answers.md` объясняет смысл ограничений, а не только перечисляет синтаксис.

Не требуется:

- проектировать промышленную миграцию;
- анализировать блокировки DDL;
- добавлять индексы кроме тех, что возникают из ключей и уникальности.
