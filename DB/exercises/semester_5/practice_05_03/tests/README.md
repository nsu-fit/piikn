# Контракт проверки

## Автоматические проверки

- Проверить наличие `README.md`, `relationship_checks.sql`, `er.md`, `relationship_evidence.md`, `answers.md`.
- Выполнить `relationship_checks.sql`.
- Проверить, что файл содержит минимум 4 комментария `-- check:`.
- Проверить, что запросы с `LIMIT` имеют `ORDER BY`.

## Hidden checks

- На этой неделе hidden checks не используются для оценки SQL-навыка, потому что студенты не пишут SQL самостоятельно.
- Возможна проверка, что `relationship_checks.sql` выполняется на базе `6 months`.

## Ручная проверка

- ER-фрагмент и `relationship_evidence.md` проверяются вручную.
- Ключевой критерий: студент должен объяснить, чем подтверждена каждая связь.
- На защите преподаватель выбирает одну связь и просит показать evidence.
