# Ожидаемые свойства решения

`queries.sql` должен содержать 7 исполняемых запросов:

- `q01` использует фильтрацию, сортировку и `LIMIT`;
- `q02` соединяет `routes` и `flights`;
- `q03` соединяет `tickets`, `segments` и `flights` через корректную цепочку;
- `q04` использует `GROUP BY` и агрегаты;
- `q05` считает выручку на уровне маршрута, а не на уровне отдельного сегмента;
- `q06` использует anti-join;
- `q07` группирует билеты по бронированию.

`answers.md` должен объяснять типовую ошибку случайного декартова произведения.

Допустимы разные корректные формы запроса, если результат переносим между размерами демобазы.
