# Контракт проверки

Автоматически можно проверить:

- наличие `README.md`, `index.md`, `selected_artifacts.md`;
- наличие ссылок на обязательные темы;
- наличие SQL-файлов для SQL-артефактов;
- отсутствие пустых ссылок и очевидно битых локальных путей.

Вручную проверяются:

- способность студента объяснить выбранные артефакты;
- воспроизводимость одного короткого сценария;
- качество ответов по ограничениям решения;
- отсутствие критических ошибок безопасности и целостности.
