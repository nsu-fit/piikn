# Ожидаемые свойства портфолио

Корректное портфолио:

- имеет понятный входной `README.md`;
- содержит индекс обязательных тем;
- хранит SQL, Markdown-ответы и проверочные сценарии рядом с соответствующими работами;
- указывает порядок воспроизведения;
- содержит 3-5 выбранных артефактов для защиты;
- не подменяет SQL скриншотами.

Слабые или невоспроизводимые работы можно включать только с явным описанием ограничения и плана исправления.
