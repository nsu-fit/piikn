# Контракт проверки

Автоматически можно проверить:

- наличие `README.md`, `answers.md`, `er.md`;
- отсутствие обязательных файлов с самостоятельно написанным SQL;
- наличие в `answers.md` разделов про таблицы, ключи, связи, бизнес-правила и диагностику;
- наличие Mermaid ER-блока или текстовой схемы в `er.md`.

Вручную проверяется:

- корректность предметной интерпретации;
- различение явных ограничений и гипотез;
- качество объяснения готовых диагностических выводов;
- короткий устный ответ по одному ключу, одной связи и одному диагностическому выводу.
