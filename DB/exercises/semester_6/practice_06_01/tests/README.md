# Контракт проверки

Автоматически проверяются:

- наличие `README.md`, `init.sql`, `seed.sql`, `checks.sql` и способа запуска;
- успешное выполнение `seed.sql`;
- наличие таблиц `app.events`, `app.orders`, `app.documents`;
- ненулевое количество строк в seed-таблицах;
- отсутствие требования выполнять прикладные запросы от суперпользователя.

Вручную проверяются:

- воспроизводимость запуска;
- корректность ролей и прав;
- ясность README и предупреждений об удалении данных.
