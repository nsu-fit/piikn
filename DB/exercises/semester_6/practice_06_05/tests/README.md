# Контракт проверки

Автоматически проверяются:

- наличие обязательных файлов;
- наличие `EXPLAIN (ANALYZE, BUFFERS)` в `before.sql` и `after.sql`;
- наличие одного основного изменения в `after.sql`;
- отсутствие запрещенных внешних подсказок или ссылок в артефакте контрольной.

Вручную проверяются:

- чтение плана;
- качество гипотезы;
- корректность сравнения до/после;
- объяснение побочных эффектов.
