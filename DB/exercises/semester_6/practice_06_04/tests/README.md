# Контракт проверки

Автоматически проверяются:

- наличие `before.sql`, `indexes.sql`, `after.sql`, `report.md`;
- наличие `EXPLAIN (ANALYZE, BUFFERS)` до и после;
- наличие минимум двух `CREATE INDEX`;
- выполнение SQL без ошибок на seed dataset;
- отсутствие изменения данных в `before.sql` и `after.sql`.

Вручную проверяются:

- связь индекса с запросом;
- корректность интерпретации планов;
- объяснение trade-off.
