# Ожидаемые свойства решения

В корректной работе:

- before/after планы сняты на одном и том же запросе;
- индекс соответствует предикату и сортировке;
- JSONB-индекс выбран осознанно: expression или GIN;
- отчет не сводится к "стало быстрее", а объясняет почему;
- указаны побочные эффекты индексов.
