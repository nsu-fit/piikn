# Контракт проверки

Автоматически проверяются:

- наличие `adr.md`, `graph-model.md`, `traversal.sql` или `traversal.md`;
- наличие минимум трех альтернатив;
- наличие traversal-сценариев;
- наличие раздела consequences/risks.

Вручную проверяются:

- качество ADR;
- обоснованность выбора модели;
- ответы на checkpoint review.
