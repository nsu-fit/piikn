# Ожидаемые свойства решения

В корректной работе:

- ADR отвечает на конкретную инженерную проблему;
- traversal-запросы описаны явно;
- реляционная альтернатива рассмотрена честно;
- graph DB выбирается только при обоснованной потребности;
- последствия для эксплуатации указаны отдельно от преимуществ модели.
