# Ожидаемые свойства решения

В корректной работе:

- partition key связан с retention и типовыми запросами;
- есть пример DDL или обоснованный design-only вариант;
- retention описан как эксплуатационная процедура;
- ADR сравнивает альтернативы и цену решения;
- индексы на партициях обсуждены явно.
