# Ожидаемые свойства решения

В корректной работе:

- backup и restore описаны командами, а не общими словами;
- restore выполняется в отдельную безопасную цель;
- smoke checks сравнимы между source и restore;
- recovery notes фиксируют ограничения логического backup;
- destructive команды выделены отдельно и помечены.
