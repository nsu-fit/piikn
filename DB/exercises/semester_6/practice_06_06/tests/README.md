# Контракт проверки

Автоматически проверяются:

- наличие обязательных файлов;
- наличие `pg_dump` и команды восстановления;
- наличие smoke checks по трем таблицам;
- отсутствие команд, которые по умолчанию удаляют source database без явного предупреждения.

Вручную проверяются:

- воспроизводимость runbook;
- безопасный restore target;
- осмысленность recovery notes.
