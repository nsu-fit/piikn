# Контракт проверки

Автоматически проверяются:

- наличие обязательных файлов;
- наличие минимум трех вариантов в trade-off table;
- наличие разделов про consistency, retention, backup/restore и operations;
- наличие access policy sketch.

Вручную проверяются:

- качество аргументации;
- реалистичность access policy;
- связь выбора с эксплуатацией.
