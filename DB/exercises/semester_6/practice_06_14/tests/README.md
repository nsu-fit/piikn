# Контракт проверки

Автоматически проверяются:

- наличие обязательных файлов;
- наличие меток `q01` - `q04`;
- выполнение `queries.sql` без ошибок, если используется seed dataset;
- наличие `OVER (` для window-запроса;
- наличие слов `materialized view`, `refresh`, `staleness`.

Вручную проверяются:

- качество модели витрины;
- понимание OLTP/OLAP различий;
- полнота quality notes.
