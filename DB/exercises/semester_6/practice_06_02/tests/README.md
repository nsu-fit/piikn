# Контракт проверки

Автоматически проверяются:

- наличие обязательных файлов;
- наличие меток `q01` - `q06` в `catalog_queries.sql`;
- выполнение запросов к системным каталогам без ошибок;
- наличие `pg_relation_filepath`, `pg_stat_activity`, `pg_total_relation_size`.

Вручную проверяются:

- корректность `cluster-map.md`;
- реалистичность MVCC/storage-наблюдения;
- отсутствие опасных операций с data directory.
