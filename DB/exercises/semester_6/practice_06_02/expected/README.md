# Ожидаемые свойства решения

В корректной работе:

- SQL к системным каталогам выполняется повторно;
- карта кластера соответствует реальному окружению студента;
- relation files связаны с конкретными таблицами;
- storage-наблюдение отделяет факт измерения от объяснения;
- явно указано, что файлы кластера нельзя редактировать вручную.
