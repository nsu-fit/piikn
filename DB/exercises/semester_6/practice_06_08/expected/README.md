# Ожидаемые свойства решения

В корректной работе:

- checklist можно использовать для повторной проверки контура;
- health check проверяет работоспособность SQL, а не только сеть;
- slow query baseline содержит конкретный запрос и план;
- роли не сведены к одному суперпользователю;
- checkpoint-демонстрация зафиксирована в `demo-result.md`.
