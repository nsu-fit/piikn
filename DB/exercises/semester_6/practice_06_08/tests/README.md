# Контракт проверки

Автоматически проверяются:

- наличие обязательных файлов;
- наличие разделов checklist;
- выполнение `checks.sql` без ошибок в учебной базе;
- наличие `EXPLAIN (ANALYZE, BUFFERS)`;
- наличие `demo-result.md`.

Вручную проверяются:

- полнота checklist;
- смысл health checks;
- качество checkpoint-демонстрации;
- реалистичность открытых рисков.
