# Ожидаемые свойства решения

В корректной работе:

- search design привязан к конкретному пользовательскому сценарию;
- source of truth не смешан с derived index;
- SQL-прототип или design sketch проверяем;
- качество поиска оценивается на тестовых запросах;
- privacy/security/staleness риски указаны явно.
