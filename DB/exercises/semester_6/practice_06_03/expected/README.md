# Ожидаемые свойства решения

В корректной работе:

- измерения выполняются SQL-запросами, а не скриншотами;
- размеры таблиц, индексов и total relation не смешиваются;
- baseline-планы приложены до создания новых индексов;
- индексные гипотезы связаны с конкретными предикатами и сортировкой;
- в отчете указана цена индекса для записи и хранения.
