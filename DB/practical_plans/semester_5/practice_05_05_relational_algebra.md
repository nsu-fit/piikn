# Практическое занятие 5.5. Реляционная алгебра без самостоятельного SQL

Место в курсе: занятие закрепляет реляционную алгебру до начала самостоятельного написания SQL-запросов.

Тип занятия: практика.

## Цели

- Записывать прикладные задачи как выражения реляционной алгебры.
- Вычислять результат выражения на малых отношениях.
- Сравнивать эквивалентные формы выражений.
- Подготовиться к последующему переводу реляционной алгебры в SQL.

## Входные условия

- Студент понимает операции выборки, проекции, соединения и разности.
- Преподаватель выдает малые табличные фрагменты и набор прикладных задач.
- Самостоятельное написание SQL на этом занятии не требуется.

## План занятия

1. Разбор табличного фрагмента: отношения, атрибуты, домены.
2. Запись задач через selection, projection и join.
3. Вычисление результата выражения на малом наборе строк.
4. Разбор разности и антисоединения как способов выразить отсутствие связанной строки.
5. Разбор деления как прикладного шаблона "для всех".
6. Сравнение двух эквивалентных выражений по результату.
7. Фиксация мест, где будущий SQL будет отличаться от классической алгебры: дубликаты, `NULL`, порядок строк.

## Задание

Студент должен решить 5-7 задач реляционной алгебры и для 2 заданий предложить альтернативную форму выражения.

Обязательные элементы:

- выражения с selection, projection и join;
- выражение с разностью или отсутствием связанной строки;
- выражение с формой "для всех";
- вручную полученный результат для 2-3 малых табличных фрагментов;
- краткое объяснение одной эквивалентности.

## Артефакт сдачи

Папка `semester-5/practice-05-relational-algebra`:

- `answers.md` с выражениями и пояснениями;
- `result_tables.md` с результатами ручного вычисления на малых отношениях.

## Критерии зачета

- Выражения соответствуют поставленным задачам.
- Результаты ручного вычисления согласованы с исходными таблицами.
- Альтернативные формы действительно эквивалентны на заданных данных.
- Нет случайных декартовых произведений.

## Контрольные вопросы: что вы должны уметь после занятия

1. Чем selection отличается от projection?
2. Как соединение связано с декартовым произведением?
3. Как выразить отсутствие связанной строки средствами реляционной алгебры?
4. Для каких задач нужна операция деления?
5. Как проверить эквивалентность двух выражений на малых данных?
6. Почему будущий SQL не всегда совпадает с классической реляционной алгеброй один к одному?
