# Решение 5.9. Минимальный DDL и ограничения

## `schema.sql`

```sql
CREATE SCHEMA IF NOT EXISTS course_lab;

DROP TABLE IF EXISTS course_lab.issue_comments;
DROP TABLE IF EXISTS course_lab.issue_tickets;
DROP TABLE IF EXISTS course_lab.issue_projects;

CREATE TABLE course_lab.issue_projects (
    project_id integer GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    code text NOT NULL UNIQUE,
    name text NOT NULL,
    CONSTRAINT issue_projects_code_format_chk
        CHECK (code ~ '^[A-Z0-9_]{2,12}$')
);

CREATE TABLE course_lab.issue_tickets (
    ticket_id integer GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    project_id integer NOT NULL REFERENCES course_lab.issue_projects(project_id),
    title text NOT NULL,
    status text NOT NULL DEFAULT 'open',
    priority integer NOT NULL DEFAULT 3,
    external_key text UNIQUE,
    created_at timestamp NOT NULL DEFAULT current_timestamp,
    closed_at timestamp,
    CONSTRAINT issue_tickets_status_chk
        CHECK (status IN ('open', 'in_progress', 'closed')),
    CONSTRAINT issue_tickets_priority_chk
        CHECK (priority BETWEEN 1 AND 5),
    CONSTRAINT issue_tickets_closed_at_chk
        CHECK (
            (status = 'closed' AND closed_at IS NOT NULL)
            OR
            (status <> 'closed' AND closed_at IS NULL)
        )
);

CREATE TABLE course_lab.issue_comments (
    comment_id integer GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    ticket_id integer NOT NULL
        REFERENCES course_lab.issue_tickets(ticket_id)
        ON DELETE CASCADE,
    author text NOT NULL,
    body text NOT NULL,
    created_at timestamp NOT NULL DEFAULT current_timestamp
);
```

## Пояснения

- `external_key` уникален, но nullable: внешняя система может еще не присвоить ключ.
- `closed_at` связан со статусом через `CHECK`, чтобы не было открытых задач с датой закрытия и закрытых задач без даты.
- Комментарий связан FK с задачей, потому что комментарий без задачи не имеет предметного смысла.
- Миграционные темы вроде online schema change, lock impact и rollback strategy остаются на 6-й семестр.
