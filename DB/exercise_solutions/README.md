# Преподавательские решения к упражнениям по дисциплине "Базы данных"

Этот каталог не предназначен для включения в студенческий шаблон репозитория.

Структура зеркалирует `DB/exercises`:

```text
exercise_solutions/
  semester_5/
    practice_05_01/
      solution.md
      example_artifacts.md
  semester_6/
    practice_06_01/
      solution.md
      example_artifacts.md
```

Формат решения зависит от типа задания:

- для SQL-практик приводится эталонный или один из допустимых SQL-вариантов;
- для контрольных приводится пример правильной структуры ответа и критерии приемлемых альтернатив;
- для проектных и ADR-заданий приводится reference answer, а не единственно возможное решение.

Если упражнение имеет индивидуальные варианты, решение фиксирует default-вариант и поясняет, какие части меняются при параметризации.

`example_artifacts.md` показывает конкретный пример ожидаемой студенческой сдачи: дерево файлов и фрагменты содержимого `README.md`, `answers.md`, SQL, runbook, ADR или других артефактов. Эти примеры не нужно отдавать студентам вместе с заданиями.
